Vaga para Desenvolvedor Front-end

Estamos procurando um desenvolvedor Front-end altamente qualificado para se juntar à nossa equipe. O candidato ideal terá experiência em desenvolvimento web e será responsável pelo design e desenvolvimento da interface do usuário de nossos aplicativos web.

Responsabilidades

  • Desenvolver e manter a interface do usuário de nossos aplicativos web
  • Trabalhar em estreita colaboração com o time de design para garantir que a interface do usuário seja atraente e fácil de usar
  • Colaborar com outros membros da equipe de desenvolvimento para garantir que os aplicativos sejam escaláveis e de alta qualidade

Requisitos

  • Experiência com desenvolvimento web Front-end
  • Conhecimento em HTML, CSS, JavaScript e jQuery
  • Capacidade de trabalhar em equipe e se comunicar efetivamente
  • Excelentes habilidades de resolução de problemas e pensamento crítico
